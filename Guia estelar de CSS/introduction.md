#Introdução

## O que significa CSS?

* Cascading Style Sheet
* Código para criar estilos no HTML
* HTML é a estrutura, CSS é a beleza visual
* Não é uma linguagem de Programação
* É uma linguagem style sheet

## Vamos ao exemplo

<!-- Use o codepen.io -->

h1 {
    color: red;
    font-size: 20px;
    background: white;
}
