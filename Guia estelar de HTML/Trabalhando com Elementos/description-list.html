Hardware do Computador

Elementos

Placa-Mãe
É o principal componente de um computador, pois é nela que todos os demais componentes e periféricos se conectam e transmitem suas informações. Além disso, será ela a responsável por definir qual tipo de processador usar, assim como placa de vídeo, memória, etc.
Hard Disk (Disco rígido)
De um tempo pra cá os usuários ganharam uma nova opção na hora de escolher a unidade responsável pelo armazenamento de seus conteúdos. Além dos HDs, agora é possível optar por dispositivos SSD, que nada mais é do que uma evolução da antiga tecnologia.
